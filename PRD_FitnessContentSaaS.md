# PRD — 오토사장 (AutoCEO Milestone SNS 자동화)
> 버전 0.2 | 2026-04-28 | 작성: BigStone Group

---

## 1. 서비스 개요

### 1.1 브랜드
- **서비스명:** 오토사장
- **상위 브랜드:** AutoCEO Milestone
- **포지셔닝:** AutoCEO Milestone의 SNS 자동화 SaaS 툴
- **슬로건:** "내 얼굴, 내 목소리로 — 텍스트만 입력하면 릴스 완성"

### 1.2 한 줄 정의
> 피트니스·의료 전문가가 **자기 얼굴과 목소리로** 전문 콘텐츠 영상을 텍스트 입력만으로 만들 수 있는 AI 영상 제작 SaaS

### 1.2 핵심 가치
- 촬영 없이 → 내 아바타가 대신 말한다
- 편집 없이 → AI가 자막·컷편집을 처리한다
- 전문성 → 의료·피트니스 특화 포맷으로 신뢰도를 높인다

### 1.3 타겟 사용자
| 구분 | 설명 |
|---|---|
| 1차 타겟 | 피트니스 트레이너, 물리치료사, 스포츠 재활 전문가 |
| 2차 타겟 | 헬스센터 운영자, 의료·건강 콘텐츠 크리에이터 |
| 채널 | trainer.milestone / Biz Auto System 수강생 |

## 3. 콘텐츠 포맷 정의

---

### Format A — 내 얼굴 릴스

**개념**
> 사용자 사진 1장 + 목소리 샘플 → AI 아바타가 텍스트를 읽어주는 영상

**사용 시나리오**
트레이너가 "오늘의 운동 팁" 스크립트를 입력 → 자신의 얼굴과 목소리로 말하는 릴스 영상 완성

**화면 구성**
```
┌────────────────────────────┐
│                            │
│     아바타 풀샷 또는 반신    │
│     (배경 단색 or 그라데이션) │
│                            │
└────────────────────────────┘
```

**기능 요구사항**

| # | 기능 | 설명 |
|---|---|---|
| A-01 | 사진 업로드 | JPG/PNG, 최대 10MB, 정면 얼굴 권장 |
| A-02 | Photo Avatar 생성 | HeyGen Photo Avatar API 호출, 최초 1회 |
| A-03 | 목소리 샘플 업로드 | WAV/MP3, 30초~3분 권장 |
| A-04 | Voice Clone 생성 | HeyGen Voice Clone API 호출, 최초 1회 |
| A-05 | 스크립트 입력 | 텍스트 직접 입력, 최대 500자 (약 1분 분량) |
| A-06 | 음성 파일 업로드 | WAV/MP3 직접 업로드 (TTS 대신 사용) |
| A-07 | 영상 생성 요청 | HeyGen Video Generate API 호출 |
| A-08 | 생성 상태 폴링 | 10초 간격 상태 확인, 완료 시 알림 |
| A-09 | 영상 미리보기 | 인앱 플레이어 |
| A-10 | 플랫폼별 다운로드 | 인스타 릴스 / 유튜브 쇼츠 / 틱톡 선택 후 다운로드 |

**출력 스펙**
| 항목 | 값 |
|---|---|
| 해상도 | 1080 x 1920 (9:16) |
| 길이 | 60초 이내 |
| 포맷 | MP4 |

### Format B — 자료 해설 릴스

**개념**
> 인포그래픽·슬라이드·관련 영상을 배경에 깔고, 우측 하단에 아바타 상체가 오버레이되어 설명하는 포맷

**사용 시나리오**
"척추측만증 예방 스트레칭 5가지" 슬라이드를 배경으로 → 아바타가 각 동작을 설명하는 전문가 콘텐츠

**화면 구성**
```
┌─────────────────────────────┐
│  [배경: 인포그래픽 / 영상]   │
│                             │
│                 ┌─────────┐ │
│                 │  아바타  │ │
│                 │  상체   │ │
└─────────────────┴─────────┘─┘
                우측 하단 오버레이
```

**기능 요구사항**

| # | 기능 | 설명 |
|---|---|---|
| B-01 | 내 얼굴 릴스 아바타 재사용 | 기존 생성된 아바타/보이스 ID 연동 |
| B-02 | 배경 콘텐츠 업로드 | 이미지(JPG/PNG) 또는 영상(MP4) 업로드 |
| B-03 | 배경 영상 직접 제작 | 슬라이드 텍스트 입력 → AI 인포그래픽 생성 (추후) |
| B-04 | 아바타 오버레이 설정 | 위치(우측하단 고정), 크기(화면 30% 기본) |
| B-05 | 아바타 배경 제거 | 크로마키 or AI 배경 제거 처리 |
| B-06 | 스크립트 입력 | 텍스트 입력 또는 음성 파일 업로드 |
| B-07 | 영상 합성 및 렌더링 | 배경 + 아바타 오버레이 합성 |
| B-08 | 자막 자동 생성 | Whisper API로 자동 자막 |
| B-09 | 플랫폼별 다운로드 | 릴스 / 쇼츠 / 틱톡 |

**출력 스펙**
| 항목 | 값 |
|---|---|
| 해상도 | 1080 x 1920 (9:16) |
| 길이 | 60초 이내 |
| 포맷 | MP4 |

### Format C — 촬영본 자동편집

**개념**
> 헬스장에서 직접 촬영한 운동 영상을 업로드 → AI 자동 자막 + 풀 편집 → 릴스 최적화 출력

**사용 시나리오**
"데드리프트 자세 교정" 촬영본 업로드 → 자막 자동 생성 + 하이라이트 컷편집 + BGM 추가 → 릴스 완성

**기능 요구사항**

| # | 기능 | 설명 |
|---|---|---|
| C-01 | 영상 업로드 | MP4/MOV, 최대 500MB, 최대 10분 원본 |
| C-02 | 자동 자막 생성 | OpenAI Whisper API, 한국어/영어 지원 |
| C-03 | 자막 스타일 선택 | 폰트/크기/색상/위치 프리셋 3종 이상 |
| C-04 | 자막 수동 편집 | 타임라인에서 텍스트·타이밍 수정 |
| C-05 | 자동 컷편집 | 무음 구간 자동 제거, 하이라이트 추출 |
| C-06 | 수동 컷편집 | 구간 선택 후 자르기/삭제/순서 변경 |
| C-07 | BGM 추가 | 기본 제공 BGM 라이브러리 or 직접 업로드 |
| C-08 | 텍스트 오버레이 | 제목/자막 카드 추가 |
| C-09 | 속도 조절 | 0.5x ~ 2x |
| C-10 | 9:16 크롭 | 원본 비율 자동 크롭 or 수동 조정 |
| C-11 | 렌더링 및 다운로드 | 플랫폼별 최적화 출력 |

**출력 스펙**
| 항목 | 값 |
|---|---|
| 해상도 | 1080 x 1920 (9:16) |
| 길이 | 60초 이내 (자동 트리밍 경고) |
| 포맷 | MP4 H.264 |

### Format D — AI 운동 시연

**개념**
> 포즈 가이드 이미지 + 모델/체형 레퍼런스 이미지를 업로드하면 AI가 실제 사람처럼 움직이는 운동 시연 영상을 자동 생성하는 포맷

**사용 시나리오**
"덤벨컬 동작 가이드 시트" + "남성 피트니스 모델 레퍼런스" 업로드 → 흰 배경에서 덤벨컬을 정확한 자세로 시연하는 15~20초 영상 생성 → 릴스/캐러셀 리소스로 활용

**기능 요구사항**

| # | 기능 | 설명 |
|---|---|---|
| D-01 | 포즈 가이드 업로드 | 운동 동작 시트 이미지(JPG/PNG) |
| D-02 | 레퍼런스 이미지 업로드 | 모델/체형/의상 레퍼런스 이미지 최대 9장 |
| D-03 | 프롬프트 입력 | 운동 종류, 속도, 카메라 앵글, 배경 스타일 지정 |
| D-04 | 영상 생성 | Seedance 2.0 image-to-video API 호출 |
| D-05 | 생성 상태 폴링 | 비동기 작업 상태 확인 |
| D-06 | 텍스트 오버레이 | 동작명, 횟수, 주의사항 텍스트 추가 |
| D-07 | 자료 해설 릴스 연계 | 생성 영상을 전문가 해설 배경으로 재사용 |
| D-08 | 플랫폼별 다운로드 | 릴스 / 쇼츠 / 틱톡 |

**출력 스펙**
| 항목 | 값 |
|---|---|
| 해상도 | 최대 1080p |
| 길이 | 15~20초 단위 생성 / 연결하여 60초 구성 |
| 포맷 | MP4 |


## 7. API 연동 설계

### HeyGen API 엔드포인트

| 기능 | 메서드 | 엔드포인트 |
|---|---|---|
| Photo Avatar 생성 | POST | `/v2/photo_avatar/avatar_id/generate` |
| Photo Avatar 상태 확인 | GET | `/v2/photo_avatar/generation/{id}` |
| Voice Clone 생성 | POST | `/v2/voice_clone` |
| 영상 생성 | POST | `/v2/video/generate` |
| 영상 상태 확인 | GET | `/v1/video_status.get` |

### 내부 API Route (Next.js)

| 엔드포인트 | 역할 |
|---|---|
| `POST /api/avatar/create` | Photo Avatar 생성 요청 |
| `GET /api/avatar/status/:id` | Avatar 생성 상태 |
| `POST /api/voice/clone` | Voice Clone 생성 |
| `POST /api/video/generate` | 영상 생성 요청 |
| `GET /api/video/status/:id` | 영상 렌더링 상태 |
| `POST /api/video/composite` | 자료 해설 릴스 합성 요청 |
| `POST /api/subtitle/generate` | Whisper 자막 생성 |
| `POST /api/video/generate` | AI 운동 시연 mock/Seedance 영상 생성 요청 |

> ⚠️ HeyGen API 키는 Vercel 환경변수에만 저장. 클라이언트 노출 금지.
